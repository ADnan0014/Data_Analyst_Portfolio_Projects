# Data Science Employee Salary Analysis

This project explores salary trends in the data science industry using real-world data. The goal was to understand how different factors like experience level, job role, company size, and work model (remote/on-site) impact salaries.



## Project Overview

In this project, I performed:

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Data visualization using Matplotlib and Seaborn
* Feature encoding for machine learning
* Built a simple Linear Regression model to predict salaries



##  Dataset

The dataset contains information about:

* Job titles
* Experience levels
* Employment type
* Work models (Remote, On-site, Hybrid)
* Salaries (in USD)
* Company location and size


##  Key Insights

* Salaries increase significantly with experience level
* Executive-level roles have the highest average salaries
* Remote jobs offer competitive salaries compared to on-site roles
* Larger companies tend to pay higher salaries
* Certain roles like *Analytics Engineering Manager* and *Data Science Tech Lead* are among the highest paying



##  Visualizations

The project includes multiple visualisations, such as:

* Salary distribution histogram
* Average salary by experience level
* Salary comparison by work model
* Heatmap of job title vs experience level
* Boxplots for company size vs salary



##  Machine Learning Model

* Model Used: Linear Regression
* Features: Experience level, company size, work model
* Evaluation Metrics:

  * RMSE: ~73,983
  * R² Score: ~0.07

The model provides a basic understanding of salary prediction but can be improved with more features and advanced models.
