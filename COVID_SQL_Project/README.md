# COVID-19 Data Analysis Project (SQL)

##  Overview
This project analyzes COVID-19 data using SQL Server. The goal is to understand the spread of the virus, death rates, and vaccination progress across different countries and continents.

---

##  Tools Used
- Microsoft SQL Server  
- SQL (Structured Query Language)

---

##  Dataset
The project uses two datasets:

- CovidDeaths  
- CovidVaccinations  

---

##  Key Analysis Performed

### 1. Basic Data Exploration
- Selected and viewed important columns like cases, deaths, and population  
- Filtered data to focus only on countries  

---

### 2. Death Rate Analysis
- Calculated the percentage of deaths compared to total cases  
- Helped understand how dangerous COVID-19 was in different locations  

---

### 3. Infection Rate Analysis
- Compared total cases with population  
- Found the percentage of population infected in each country  

---

### 4. Country and Continent Comparison
- Identified countries with the highest infection rates  
- Found countries and continents with the highest death counts  

---

### 5. Global Trends
- Calculated total global cases and deaths per day  
- Measured overall death percentage worldwide  

---

### 6. Vaccination Analysis
- Joined deaths and vaccination datasets  
- Calculated total vaccinations over time using rolling sums  

---

### 7. Advanced SQL Techniques
- Used CTEs (Common Table Expressions) for better readability  
- Created temporary tables for intermediate results  
- Built a view to store final results for future use
